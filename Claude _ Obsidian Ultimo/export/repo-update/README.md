## Cómo verlo

Es un archivo HTML estático, no necesita build. Dos formas:

1. **Local**: cloná el repo y abrí `Cumbre Innovacion 2026.html` en cualquier navegador moderno.
2. **GitHub Pages**: en *Settings → Pages*, elegí la branch `main` y carpeta `/ (root)`. En 1–2 minutos queda online.

### Navegación

- **Flechas ← →** o **espacio** para avanzar entre slides.
- **F** para pantalla completa.
- **P** o `Ctrl/Cmd + P` para imprimir / exportar PDF (una página por slide).
